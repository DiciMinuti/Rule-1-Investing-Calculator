# Security Policy

## Supported Versions

Security fixes are handled on the default branch.

## Reporting A Vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository if available.

If that is not available, contact the maintainer through the repository owner profile and avoid publishing exploit details until the issue is reviewed.

## Scope

Useful reports include:

- exposed secrets or unsafe environment handling
- cross-site scripting or unsafe rendering
- unsafe file handling
- dependency vulnerabilities with a practical exploit path
- issues that could corrupt or leak browser-local workspace data

This app is an educational research tool and does not execute trades, hold brokerage credentials, or store server-side user accounts.
